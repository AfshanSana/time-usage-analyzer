
# Time Usage Analyzer 📊
## Description
Time Usage Analyzer is a Python-based command-line tool that analyzes how time is spent on different activities.
It provides summaries, visualizations, and suggestions to improve productivity.

## Features 🚀
Read data from CSV files

Analyze total time spent on activities

Day-wise breakdown (for weekly data)

Pie chart visualization 📊

Smart suggestions for better time management
